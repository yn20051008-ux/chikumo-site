01 — Definition

持続的生存：最小定義

存在が継続し続けるとは、結果が次の起点を更新し、その更新が世代を越えて接続され、断絶せずに引き継がれることを指す。九式は、この"継続と更新"を成立させる最小構成を9つの漢字で定義する。

02 — Nine Elements

九式

因
いん / In

存在が発生・更新される起点。

理
り / Ri

存在が成立するための条件構造。

衡
こう / Kou

存在が破綻せず保たれる制約。

縁
えん / En

世代を跨いで存在を接続する関係。

循
じゅん / Jun

結果が次の因を更新する仕組み。

許
きょ / Kyo

循が成立し得る範囲を定める境界。

幅
はば / Haba

許を超えた後も、絶に至るまでの許容量を定める境界。

乱
らん / Ran

許を超え、幅の内側にある再現性喪失状態。

絶
ぜつ / Zetsu

幅を超え、存在と更新が断たれた状態。

03 — Structure

最小骨格

無（0）

完全排除状態。発生不能・観測不能・構造不能。

一式
因

無ではないが未確定の起点。ここから更新が開始されうる。

三式
因・理・衡

確定が成立する。因が理によって成立し、衡によって保たれる。

五式
因・理・衡・縁・循

確定が世代を越えて接続され、結果が次の因を更新する仕組みが成立する。

九式
因・理・衡・縁・循・許・幅・乱・絶

五式に運用境界（許・幅・乱・絶）を加え、持続的生存が"成立している領域／崩れた領域"を区別できるようにした定式。

04 — Boundary

運用境界

循域 許の内側。循が成立する領域。
乱域 許を超え、幅の内側。再現性が失われ、循は成立しない領域。
絶域 幅を超えた領域。断絶が確定する領域。
