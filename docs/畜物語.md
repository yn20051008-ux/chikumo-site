# 畜物語（ちくものがたり） ─ 作品の森

> 畜学の十冊目。他の九つの蔵──**十四式＝どう作るのか／遊戯学＝なぜ作るのか／設計学＝どの高さ・どの順で作るのか／畜史＝何を作ったのか／思考学＝どう考えたのか／肥料学＝なぜ終えたのか／技術学＝何で作るのか／祖元学＝そもそも、なぜこの形なのか／標準学＝何を土台に選ぶのか**──は、どれも畜物語という器の**中**で通用する規則・掟を記す。
> 畜物語だけは規則ではない。設計学（`docs/設計学.md`）でいう**森（全体＝十四式の家⑧、世界・掟・共有の臓器）**そのものを記す本──畜物語という作品世界の姿を、規則の形に切り分ける前の、一つの森のまま留めておく場所。
> 公開ページは `monogatari/index.html`（蔵書 `/Library/` の棚）。内容を変えたら両方そろえること。
> 内容は思いついたときに、正論蒔宿との対話から追記していく。いま巻は空である──空であることが、この本の現在の姿として正しい。
>
> 起：丙午2026・文月七、正論蒔宿との対話より結ぶ。
