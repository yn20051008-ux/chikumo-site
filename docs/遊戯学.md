## 七相（要旨。詳細と「封」は `yugigaku/index.html`、運用の鍵は `docs/畜学.md`）
1. **壱 既視の禁** … 既存ジャンルの焼き直しを作らない。この世にまだ無い遊びを発明する。
2. **弐 焔の形** … 短く激しく必ず終わる超中毒（60秒で焼き切る／無限ループ＝底なし＝禁忌）。
3. **参 終を售る** … 終わった瞬間＝人生に向き合う瞬間を設計で発火。終いの無音を罰でなく祝福に。
4. **四 万象同時の醒** … 世界ランキングは順位争いでなく、世界が同時に顔を上げる祭壇。
5. **五 二親の的** … 遊び手は親子。音・光・BGMは飾りでなく二人の記憶を濃くする本体。
6. **六 敵は逆縁のみ** … 敵はブラックこっこ＝自分の影＝逆縁（縁＝衡＋∿衡−を断つ力）だけ。相手を敵にしない。
7. **七 一器に納む** … HTML一枚に自己完結。外部は世界ランキングのFirebaseのみ。インラインJS編集後は `node --check`。

> 補：⑥の「縁」は十四式 ⑦ に定義される（縁＝衡＋ ∿ 衡−）。逆縁＝その ∿ の切断。
