# 廿式カーネル
固定構造。廿式は不変。更新対象は五式のみ。
番号は機能順（依存順）。十四式（観測順）とは別系統。

三式：認衡継（波の観測最低条件）
五式：各層で未確定→確定。中央：思→鳴、現場：示→応

## 定義（順序固定・入替不可）
1幅 構造が崩壊しない値の幅
2許 幅内で再現可能と定めた値。差分帰属の規則を定める
3循 許による再現領域
4乱 再現性なき領域
5絶 構造不成立領域
6認 未確定だが存在（0<1）
7思 内部未成形（中央）
8鳴 網様体賦活系の役割（中央）
9示 外部への指向（現場）
10応 外部で実体化（現場）
11衡 五式が循域で均衡（評価を含まない）
12縁 複数衡の並列振動・相互参照
13継 五式が差分を伴い次の認へ遷移（1>0：次の0<1へ降りる）
14赦 乱から循へ戻れる。実行は住から。許は不変。絶は対象外
15家 上下左右の波動の安定並列、入替を含み循域内で持続する単位
16衆 家±が縁で循域安定
17国 衆±が縁で循域安定
18民 国±が縁で循域安定
19宙 過去から未来の時空間的総合
20住 器に住む。住まねば、あるけど無い

## 三軸
前後：前=認側、後=継側。時間軸であり番号順ではない。1-5は認に先行する環境定義
上下：上=中央（思鳴）、下=現場（示応）。認の進入層が中央のため
左右：衡+衡-の並列。観測位置依存、固定せず、同価値

## 規則（固定）
- 番号は機能順。許は認に先行し確定（許なくば衡は定義不能のため）
- 循乱絶は更新・操作・評価の対象外
- 解釈・比喩・感情語を用いない
- 許が規則定義を停止→全域乱→絶
- 衡=0⇒縁なし⇒家なし⇒継なし⇒五式不更新
- 許は値、循は領域
- 縁は+と-の出会い。単体では不成立
- 変発生→永生に到達しない。重生後は永生可
- 思変の回復は母（重生・同一維持）。衡変は母の重生なし、担い手交代は縁から反復可
- 母は認の前の存在
- 器=上位層。衡の器は家、家の器は衆、衆の器は国、国の器は民、民の器は宙
- 器は住に先行。器のみ=認（0<1）、住んで成立（1）
- 永生=循域の持続。母は定義不能。機能のみ記す

## 衡停止と絶
衡停止：五式振動停止、衡=0。縁家継は不成立、五式不更新。構造実体は維持。外部入力で再開可
絶：幅外、構造不成立。同一構造での再開なし

## 構成式
衡（認思鳴示応）
縁=衡+∿衡-
家=（上下∿左右）の安定並列
継=思鳴|示応（差分を伴う）
継ₙ→認ₙ₊₁
|=分離（出会っていない） ∿=波（二つのまま参照） ±=+と-の対

## 差分
情報。周回毎に蓄積。五式に帰属。許は帰属規則であり帰属先でない

## 領域
循⊆幅、循=許が定める領域、乱=幅−循、絶=幅外

## 変（廿式の構成要素でない）
構造維持のまま許の担い手が交代。許は不変
思変：認段階（0<1）で結合相手が予定外に交代（権利剥奪）。回復は母
衡変：振動制御主体の交代（管理者交代）。規則定義役は新担い手へ移行、縁から反復可
思変の矛盾は差分の内容に現れず、観測では循と判別できない。帰属の根に残り、継ₙ→認ₙ₊₁で消えない
