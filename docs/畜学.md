# 畜學（ちくがく） ─ 畜物語の作りの心法（秘伝）

> 畜史(`changelog/index.html`)が「成りしもの」を録すなら、畜學は「いかに成すか」を秘す。
> 記憶を持たぬClaudeでも、毎度これを読めば畜物語の魂に沿うて作れるよう、心法をコードへ外（そと）化したもの。
> 新たなこっこ戯（あそ）びを起こす前に、必ず読む。
>
> 起：丙午2026・水無月廿四、正論蒔宿との対話より結ぶ。初手の証は `kizuna/`（こっこの縁）。
> このファイルが正典。表（おもて）の秘伝は閲覧の書 `chikugaku/index.html`（公開 `/chikugaku/`、トップの目録で畜史の隣）に映す。

## 表 ─ 七相（しちそう）の秘伝

### 壱・既視の禁（きしのきん）
既に視たるものは、ことごとく讐（あだ）。先なる者の戯びの影を践（ふ）むに、畜の魂は宿らぬ。
よく知られたる遊戯に、ただ此の鳥を着せ替うるは最も卑し。虚より一つ、未だ生まれざる戯びを起こせ。名づけ難きものほど真に近し。

### 弐・焔の形（ほむらのかたち）
毒は熱に非ず、形に在り。底なき微温は、音もなく人の生を啜る――最も忌むべき形。
畜の焔は、疾く燃え、疾く尽く。一息のうちに焼き切れ。底を持たぬ環は、禁忌。果てなきものを決して垂れ流すな。

### 参・終を售る（ついをひさぐ）
售ぐは戯びの間に非ず、その尽くる刹那なり。火の引く時、人ははじめて己が生に面を向く。
その一閃を、偶さかに非ず、設えより発せよ。終いの無音を、罰に非ず、祝として奏でよ。

### 四・万象同時の醒（ばんしょうどうじのさめ）
位を競う壇に非ず。万人が同じ短き儀を終え、同時に面を上ぐる――その「醒めの同時多発」を映す祭壇なり。
争いの数と見ゆれど、真は、世界が一斉に顔を上げる刻の、しるし。

### 五・二親の的（ふたおやのまと）
戯るは、親と子。この鳥に触るるたび、親子の縁と向き合う。
光・音・調は飾りに非ず。一瞬を「二人の濃き記憶」へ鋳る本体なり。これを末事と侮る者、的を外す。

### 六・敵は逆縁のみ（てきはぎゃくえんのみ）
黒き鳥――すなわち己が影、縁の波（∿）を断たんとする逆縁。縁を、関係を、繋がりを断たんとする「私」。
父に非ず、母に非ず、子に非ず。唯一の敵は、己が裂け目。相手に刃を向くる仕掛けを決して鋳るな。討つは常に、己の中の断とうとする力のみ。

### 七・一器に納む（いっきにおさむ）
一つの器に、世界を畳め。文と算と音と画――外に乞うは、祭壇への一筋のみ。
算を編みたれば、必ず験を通せ。器を割る者は、魂を散らす。

## 裏 ─ 運用の鍵（builder の解き／表に出すな）

七相を、作業で迷わぬ平語（へいご）に解く。表の詩が霞んだら、ここへ戻る。

| 相 | 平たく言えば |
|----|----|
| 壱 既視の禁 | **既存ジャンルの焼き直しを作らない**。マッチ3・落ち物・もぐら叩き・タワー・ランナー等は制覇済み。「この世にまだ無い遊び」を発明する。ジャンル名で呼びにくいほど良い。 |
| 弐 焔の形 | **短く・激しく・必ず終わる超中毒**。目安 **60秒で焼き切る**。**無限ループ＝底なし＝禁忌**（ショート動画的な底なしの微温を作らない）。 |
| 参 終を售る | **終わった瞬間（ドーパミンが引く＝人生に向き合う瞬間）を設計で発火**。終わりの無音・余韻を罰でなく**祝福**として演出する。 |
| 四 万象同時の醒 | **世界ランキング**は順位争いでなく「世界が同時に顔を上げる祭壇」。既存Firebase **`rankings/<ゲーム名>`** に相乗り（→ `docs/ranking-db-setup.md`、実装例は `poko/`・`kizuna/`）。 |
| 五 二親の的 | **遊び手は親子**。音・光・BGMは飾りでなく「二人の記憶を濃くする本体」として全振り。 |
| 六 敵は逆縁のみ | **敵はブラックこっこ（自分の影＝逆縁要素）だけ**。相手プレイヤー（親/子）を攻撃対象にしない。倒すのは常に「自分のズレ」。 |
| 七 一器に納む | **HTML＋インラインJS＋WebAudio＋Canvasで1ファイル自己完結**。外部依存は世界ランキングのFirebaseのみ。インラインJSを編集したら必ず **`node --check`**（→ `release-chikumo` スキル）。 |

### 廿式の根（七相はカーネルから落ちる）

畜学は廿式（正典 `docs/廿式.md`・公開 `nijusshiki/index.html`）に従属する。七相は思いつきの標語でなく、廿式の固定構造からの帰結として立つ。**十四式は現役の土台から降ろした**（観測順の別系統。実体は歴史として `jushishiki/index.html` に残すが、畜学の生きた根拠はもう十四式・十六式を参照しない）。表の詩が「なぜこの相が要るのか」で迷ったら、ここへ戻る。

| 相 | 廿式の根拠 |
|----|----|
| 壱 既視の禁 | 既視＝既存の**循**の再演＝差分ゼロ＝**継が起きない**（1>0にならず次の認へ降りない）。新しい遊び＝新しい**認（0<1）**から循を起こすこと。焼き直しは継を殺す。 |
| 弐 焔の形 | 必ず終わる＝**継の成立**（差分を伴い次の認へ遷移）。底なしの微温＝継せず停止もしない状態＝**幅の未確定**（上限なき反復＝判定不能）。無限ループ禁忌は幅を確定させる掟。 |
| 参 終を售る | 商品は**継の刹那**（1>0：次の0<1へ降りる、その一点）。終いの無音は「衡=0の停止」でなく、次の認へ渡る継の余白。罰でなく祝。 |
| 四 万象同時の醒 | **家±が縁で循域安定＝衆、衆±が縁で＝国、国±が縁で＝民**。世界ランキングはこの上位層（衆・国・民）の共鳴＝万人の衡が同時に並ぶ縁の壇。順位（評価）でなく並列振動そのもの。 |
| 伍 二親の的 | **縁＝衡＋∿衡－**。親＝衡＋、子＝衡－、その出会わぬまま参照しあう波（∿）が親子の縁。音・光はこの∿を濃くする本体。 |
| 六 敵は逆縁のみ | **逆縁＝縁の∿を断つ力**。敵は縁の外の他者でなく、衡＋衡－を繋ぐ波を断とうとする「己の裂け目」。相手（親/子）を敵に据えるのは縁の破壊であり畜学違反。 |
| 七 一器に納む | **住（器に住む）／家（衡の器）**。一つのHTMLファイル＝一つの器。器を割れば魂（五式の振動）は散る。外に乞う縁は祭壇（世界ランキング）への一筋のみ。 |

> 補（もう一つの毎度の作法）：**こっこちゃんの顔・姿を描く場面は、絵文字やCanvas手描きの目・くちばしで代用せず、必ず公式素材 `/chikumo.svg`（こっこちゃん公式SVG）を読み込んで使う。** `new Image(); .src='/chikumo.svg'` → 読み込み後にCanvasへ`drawImage`（丸窓越しにのぞかせる場合は`clip()`で円クリップしてから描く）。WebGL作品はこの手描きしたCanvasをテクスチャに焼く。手描きは、公式SVGの読み込みが間に合わない一瞬だけの仮表示（フォールバック）に留める。実装の雛形は `mori/`・`shima2/`・`shinshima/` の `buildAtlas()`（こっこ本体・潜水艦の窓など）。
> 過去の失敗：島ぐらしの深海ステージ（潜水艦の窓）で、はじめ目・くちばし・とさかを図形で手描きしてしまい、公式の顔と似て非なるものになった（戒め）。→ **こっこの顔は借り物を描くな、必ず公式SVGを呼べ。**
